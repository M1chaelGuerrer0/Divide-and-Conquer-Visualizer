Divide and Conquer basic idea

The divide and conquer approach involves breaking a problem into smaller subproblems, and once the solutions to the subproblems are found they are combined to obtain a final solution.

Examples of Divide and Conquer

Some common examples of divide and conquer algorithms include binary search and merge sort.

Binary Search

Binary seach is a searching algorithm that finds the position of target element within a sorted array. The algorithm works by constantly dividing the search interval in half.

  • compare the middle element with the target element
  • if the target element is equal to the middle element, the search is complete
  • if the target element is less than the middle element, search the left half of the array
  • if the target element is greater than the middle element, search the right half of the array
  • repeat the process until the target element is found or the interval is empty

Merge Sort

Merge sort is a sorting algorithm that uses the divide and conquer approach to sort an array. The algorithm works by recursively dividing the array into two halves, sorting each half and then merging them back together resulting in a sorted array.
